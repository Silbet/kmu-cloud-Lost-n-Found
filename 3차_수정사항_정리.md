# 3차 수정사항

클로드 코드 전달용 수정 요청 정리입니다.

---

## 1. 검색 메뉴 - 상태 필터 추가

- 검색 메뉴에 **상태(status)에 대한 필터** 기능을 추가해 주세요.
- 기존 검색 조건과 함께 상태값으로도 필터링이 가능해야 합니다.

---

## 2. 매칭 승인 로직 버그 수정

- **버그 내용**: 하나의 분실물에 대해 매칭 후보가 여러 개 있을 때, 모든 후보에 대해 승인 요청이 가능한 오류가 있습니다.
- **수정 방향**: 하나의 분실물당 **단 하나의 매칭 후보만 승인 요청**이 가능하도록 제한해 주세요.

---

## 3. 일반사용자 - 코드 수령 후 취소 버튼 추가

- 일반사용자가 매칭 승인을 받고 **코드를 수령한 단계**에서, 해당 건을 **취소할 수 있는 버튼**을 추가해 주세요.
- 사용자가 실수로 승인 받았거나 더 이상 필요하지 않은 경우 취소 가능해야 합니다.

---

## 4. 일반사용자 메뉴 정리

- 일반사용자로 로그인했을 때 **상단/사이드 메뉴에서 아래 두 메뉴를 삭제**해 주세요.
  - `분실 신고`
  - `습득물 등록`
- 사유: 각각 `내 신고`, `내 등록` 메뉴 안에 새로 추가할 수 있는 버튼이 이미 존재하므로 메뉴 중복입니다.
- 관리자/다른 권한 사용자의 메뉴 구성에는 영향이 없도록 **일반사용자 권한에 한해서만** 적용해 주세요.
